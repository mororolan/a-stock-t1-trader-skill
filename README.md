# 📊 A股T+1超短线交易分析系统

> 基于 Claude AI Skill 的多维度短线选股与交易纪律框架

## 🎯 系统定位

这是一个专为 **"当日选股 + 尾盘买入，次日择机卖出"** 设计的超短线交易辅助分析系统，以 [Claude Custom Skill](https://docs.anthropic.com) 的形式运行。

**核心理念**：多维打分筛选 → 严格纪律执行 → 次日止盈止损 → 闭环复盘优化

⚠️ **免责声明：本系统仅提供分析框架和信息整理辅助，不构成任何投资建议。股市有风险，投资需谨慎。所有交易决策由使用者自行负责。**

## 📐 六维评分体系

对候选标的进行 6 个维度打分，每项 10 分，满分 60 分：

| 维度 | 权重 | 评估重点 |
|------|------|---------|
| 技术形态 | 25% | K线、均线系统、MACD/KDJ共振 |
| 量价关系 | 20% | 放量突破、缩量回调、换手率 |
| 板块强度 | 20% | 板块排名、联动性、龙头地位 |
| 资金博弈 | 15% | 主力净流入、龙虎榜席位 |
| 消息催化 | 10% | 催化剂明确性、预期差 |
| 基本面兜底 | 10% | PE/PB、营收增速、财务安全 |

**评级标准**：A级(48-60) → B级(36-47) → C级(25-35) → D级(<25，淘汰)

## 🔴 核心纪律（六条红线）

1. 单笔止损不超过 **-3%**，触及即刻清仓
2. 单日亏损不超过总资金 **2%**，触及后停止交易
3. **永远不把 T+1 变成 T+N** — 次日 14:30 前必须清仓
4. **永远不加仓摊低成本**
5. 大盘评级🔴时 **绝对空仓**
6. D级标的 **绝不参与**

## 🚀 安装使用

### 方式一：直接安装 .skill 文件

下载 `a-stock-t1-trader.skill` 文件，在 Claude 中上传即可使用。

### 方式二：手动部署

将整个 `a-stock-t1-trader/` 文件夹放入你的 Claude Skills 目录中。

### 使用示例

| 你说的话 | 系统执行 |
|---------|---------|
| "帮我做今天的盘前分析" | 政策面 + 消息面 + 大盘环境扫描 |
| "今天有什么可以买的" | 全流程分析，输出选股报告 |
| "帮我分析一下 600519" | 对茅台执行六维打分 |
| "我买了300750，明天怎么操作" | 输出次日止盈止损执行计划 |
| "帮我复盘今天的交易" | 输出复盘模板 |

## ⚙️ 技术说明

- 系统通过 `web_search` 实时获取政策、消息、行情数据
- 由于 Claude 无法接入实时行情 API，技术面分析依赖搜索数据或用户上传的数据
- 建议配合行情软件（同花顺/东方财富等）获取精确数据后输入系统分析
