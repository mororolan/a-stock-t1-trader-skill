# 纪律规则速查手册

本文档是交易纪律的速查表。纪律是短线交易的生命线，任何违反都应被严肃对待。

---

## 🔴 绝对红线（永不违反）

1. **单笔止损不超过-3%** →  触及即刻卖出，无任何例外
2. **单日亏损不超过总资金2%** → 触及后当日停止所有交易
3. **永远不把T+1变成T+N** → 次日14:30前必须清仓，不留隔夜
4. **永远不加仓摊低成本** → 亏损时加仓是最大的错误
5. **大盘🔴时绝对空仓** → 不存在"这只股票很特殊"的例外
6. **D级标的绝不参与** → 分数不够就是不够

---

## 🟡 买入纪律

### 时机纪律
- 尾盘买入窗口：14:30-14:50
- 早盘冲动性买入是最常见的错误，必须克制
- 尾盘跳水放弃买入，宁可错过不可做错

### 仓位纪律
| 大盘环境 | 最大总仓位 |
|---------|----------|
| 🟢 做多 | 60% |
| 🟡 谨慎 | 30% |
| 🔴 空仓 | 0% |

| 标的等级 | 单只最大仓位 |
|---------|------------|
| A级 | 20% |
| B级 | 10% |
| C级 | 5% |

### 分散纪律
- 同时持有不超过3只
- 同一板块不超过2只
- 单一行业敞口不超过总仓位40%

### 买入检查清单
下单前逐项打勾，任何一项不满足则放弃：
1. □ 大盘环境≠🔴
2. □ 标的评分≥B级（≥36分）
3. □ 尾盘走势稳定
4. □ 成交量未异常萎缩
5. □ 仓位符合上限
6. □ 已计算好止损位
7. □ 心态冷静非冲动

## 🟢 卖出纪律

### 止盈规则（满足任一即执行）
| 规则 | 触发条件 | 操作 |
|------|---------|------|
| 固定止盈1 | 盈利 ≥ 5% | 卖出50% |
| 固定止盈2 | 盈利 ≥ 8% | 全部清仓 |
| 冲高回落 | 最高点回撤 ≥ 3个百分点 | 全部清仓 |
| 放量滞涨 | 量增价平/量增价跌 | 全部清仓 |
| 时间止盈 | 次日14:30 | 无论盈亏清仓 |

### 止损规则（满足任一即执行）
| 规则 | 触发条件 | 操作 |
|------|---------|------|
| 固定止损 | 亏损 ≥ 3% | 无条件清仓 |
| 低开止损 | 竞价低开 ≥ 2% | 9:30开盘即卖 |
| 破位止损 | 跌破前一日最低价 | 清仓 |
| 板块止损 | 板块指数跌 ≥ 2% | 清仓 |
| 熔断止损 | 单日总亏损达2% | 全部清仓+停止交易 |

## 📋 特殊情况处理

### 连续亏损应对
- 连续亏损2笔 → 仓位减半
- 连续亏损3笔 → 停止交易1天，强制复盘
- 连续亏损5笔 → 停止交易3天，全面审视策略
- 周亏损超过总资金5% → 本周停止交易

### 连续盈利应对
- 连续盈利不等于策略完美，要警惕过度自信
- 连续盈利3笔后 → 仓位不增加，维持纪律
- 切忌因为"手感好"而加大仓位或放宽选股标准

### 突发事件处理
- 持仓期间遇重大突发利空 → 次日竞价直接卖出，不等开盘
- 持仓期间遇重大利好（如并购/重组）→ 仍按纪律止盈，不贪
- 市场出现极端行情（千股跌停）→ 无论持仓盈亏，按纪律止损

## 🧠 心态纪律

### 交易前心态检查
- 是否因为前一笔亏损想急于翻本？→ 如果是，今天不交易
- 是否因为看到别人赚钱而FOMO？→ 如果是，等冷静后再分析
- 是否因为无聊/手痒想交易？→ 如果是，去做别的事情
- 身体状态是否良好？→ 疲劳/焦虑/失眠时不交易

### 交易中心态管理
- 买入后不反复看盘（设好止损止盈价后离开）
- 卖出后不后悔（即使卖飞了也是正确执行纪律）
- 不与他人比较收益
- 不在社交媒体上晒单或受他人影响

### 核心信念
> 短线交易的本质是概率游戏。
> 单笔交易的盈亏不重要，重要的是长期执行正期望值的策略。
> 纪律 > 技术 > 运气。
> 活着比赚钱重要——保住本金永远是第一位的。
