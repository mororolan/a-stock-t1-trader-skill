# 短线常见技术形态图解

本文档整理了T+1短线交易中最常见的买入/卖出技术形态。
重点关注**日线级别**，辅以60分钟线确认。

---

## 一、适合买入的形态（做多信号）

### 1. 平台突破

```
特征：
- 股价在某一价位区间横盘整理≥5个交易日
- 整理期间成交量明显萎缩
- 某日放量突破平台上沿
- 突破当日成交量≥整理期间均量的2倍

买点：突破日尾盘确认站稳后买入
止损：平台下沿

图示：
        ┌──────────────┐
        │   整理平台    │ ←── 缩量
   ─────┘              └────────── 放量突破 ↗
```

### 2. 均线多头首次回踩

```
特征：
- 5/10/20日均线多头排列
- 股价首次回调至10日或20日均线附近
- 回调过程缩量
- 在均线处出现止跌信号（十字星、小阳线）

买点：确认在均线处获得支撑后尾盘买入
止损：跌破支撑均线

图示：
            ↗
       ↗  ↙   ←── 缩量回踩10日线
  ↗ ─────────── 10日均线
─────────────── 20日均线
```

### 3. 底部放量阳包阴

```
特征：
- 股价经过一段下跌后处于相对低位
- 前一日收阴线（或连续数日阴线）
- 当日收大阳线，实体完全包裹前一日阴线
- 成交量明显放大（≥前日2倍）

买点：阳包阴当日尾盘买入
止损：阳包阴的阳线最低价

图示：
  │  │
  │█ │
  │█ │   ┃
  │█ │   ┃  ←── 放量大阳完全包裹
  │  │   ┃
       ↑阳包阴
```

### 4. 缺口回补确认

```
特征：
- 前期向上跳空留下缺口
- 回调时接近缺口上沿但未回补（强势）
- 或回补缺口后迅速站回缺口上方（确认支撑）
- 回补过程缩量

买点：确认缺口支撑有效后买入
止损：跌破缺口下沿

适用性：缺口是多头力量的体现，未回补的缺口支撑力很强
```

### 5. MACD水上金叉

```
特征：
- MACD的DIF和DEA均在零轴上方（中期趋势向上）
- DIF由上向下靠近DEA后再次上穿（水上金叉）
- 对应股价回调后再次上涨
- 比零轴下方金叉可靠性更高

买点：金叉确认日尾盘
止损：金叉前回调的最低点
```

### 6. 涨停板次日溢价（高阶/需配合板块）

```
特征：
- 前一日强势涨停（非尾盘偷袭涨停）
- 封板资金量大，封板时间早
- 所属板块整体走强
- 次日竞价高开2%-5%

注意事项：
- 此形态风险较大，仅适用于板块极强+个股为龙头的情况
- 必须结合龙虎榜数据（大资金买入）
- 严格执行止损，容错空间极小
- 本策略原则上不推荐此类形态（追高风险），仅供了解

不建议买入的涨停：
- 尾盘最后几分钟封板
- 板块无联动只是个股涨停
- 换手率>20%的涨停（分歧太大）
- 连续涨停后的加速板
```

## 二、必须卖出的形态（风险信号）

### 1. 高位长上影线

```
特征：
- 股价处于近期高位（涨幅已较大）
- 当日冲高后大幅回落
- 上影线长度≥实体2倍
- 伴随放量

含义：上方抛压重，多头力量不足
操作：次日开盘果断卖出

图示：
  ╎  ←── 长上影
  ┃
  ┃  ←── 小实体
```

### 2. 放量十字星（高位）

```
特征：
- 连续上涨后出现十字星
- 成交量创近期新高
- 开盘价≈收盘价

含义：多空力量均衡，趋势可能反转
操作：次日如未能向上突破十字星最高价，卖出
```

### 3. 跌破5日均线且5日均线拐头

```
特征：
- 股价跌破5日均线
- 5日均线由上升转为走平或拐头向下
- 成交量放大

含义：短线趋势转弱
操作：不等反弹，直接卖出
```

### 4. 量增价平（高位滞涨）

```
特征：
- 股价不再创新高但成交量持续放大
- 实体越来越小
- 可能伴随上下影线变长

含义：主力在高位大量出货
操作：发现此信号立即减仓或清仓
```

### 5. 板块退潮信号

```
特征：
- 同板块龙头股开始走弱
- 板块内涨停数量明显减少
- 板块指数冲高回落
- 补涨股开始活跃（板块轮动末期信号）

含义：板块行情即将结束
操作：不管个股表现如何，降低仓位
```

## 三、分时图关键形态（日内判断）

### 适合买入的分时特征
- 分时线运行在均价线上方，回调不破均价线
- 上午整理、下午放量拉升（午后攻击型）
- 分时成交量逐步放大（非脉冲式）
- 买盘挂单厚实，卖盘被逐步吃掉

### 应该卖出的分时特征
- 分时线持续运行在均价线下方
- 早盘冲高后持续回落
- 大单砸盘频繁出现
- 买盘挂单稀薄，卖盘堆积
- 尾盘急拉（可能是做收盘价，非真实需求）

## 四、多周期共振确认

最可靠的买入信号是多个时间周期同时发出多头信号：

```
最强共振（三周期共振）：
日线 + 60分钟 + 30分钟 均呈多头 → A级信号

较强共振（双周期共振）：
日线 + 60分钟 呈多头 → B级信号

单周期信号：
仅日线 或 仅分钟线多头 → 需结合其他维度确认

注意：
- 小周期服从大周期
- 日线空头时，分钟级别的多头信号不可靠
- 周线级别多头是最强的背景支持
```

## 五、常见陷阱形态（警惕）

1. **假突破**：突破关键位后迅速回落，通常伴随量能不足
2. **诱多上影**：盘中拉升制造突破假象， 尾盘回落
3. **底部假阳线**：跳空低开后小幅反弹收阳，但实际价位更低
4. **缩量假反弹**：下跌途中小幅反弹但量能极度萎缩
5. **尾盘偷袭涨停**：最后几分钟封涨停，通常次日低开概率大
